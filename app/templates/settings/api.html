🔑 Ozon API 配置

在项目根目录的 .env 文件中配置你的 Ozon Seller API 凭证： 

OZON_CLIENT_ID=你的Client-ID
OZON_API_KEY=你的API密钥
OZON_BASE_URL=https://api-seller.ozon.ru

可在 Ozon 卖家后台 → 设置 → 卖家API 中生成密钥。
